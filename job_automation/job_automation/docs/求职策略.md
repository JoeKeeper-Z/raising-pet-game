# 个人求职策略文档

## 你的定位

**目标岗位类型**：AI辅助开发 / 初级全栈 / 游戏开发（初级）  
**核心竞争力**：善用AI工具 + 实战项目 + 快速学习能力  
**目标薪资**：7k-10k（深圳）  

---

## 优势分析

### ✅ 你的优势
1. **计算机本科学历** - 满足大部分公司的基本要求
2. **2个完整项目** - 小游戏 + 自动化工具，展示实际能力
3. **AI工具运用** - 这是趋势，很多公司开始重视
4. **薪资要求合理** - 7-10k对初级岗在深圳是合理区间
5. **可立即入职** - 全职求职，时间灵活

### ⚠️ 需要注意
1. **零工作经验** - 需要避开要求"X年经验"的岗位
2. **技术深度有限** - 面试时要诚实，强调学习能力
3. **项目有bug** - 承认并说明持续优化中

### 🎯 优先投递（匹配度高）
| 类型 | 原因 | 关键词 |

### ❌ 避免投递
- 要求"3年以上经验"的岗位
- 明确标注"资深"、"专家"、"架构师"的岗位
- 外包/劳务派遣性质的工作

## 匹配规则说明

系统会这样评估每个职位：

### 硬性过滤（一票否决）
- ❌ 薪资低于7000
- ❌ 地点不在深圳（除非你接受其他城市）
- ❌ 要求3年以上经验

### 匹配度评分（0-100分）

**关键词匹配加分**：
- "初级"、"助理"、"应届生" +15分
- "前端"、"Python"、"JavaScript" +10分/个
- "游戏"、"AI"、"工具" +10分/个
- "培养"、"转行" +20分

**减分项**：
- "3年经验"、"5年经验" -50分
- "资深"、"专家" -30分

### 处理逻辑
- **≥85分**：自动投递（很少，通常是完美匹配）
- **60-85分**：推送给你确认后投递（大部分合适岗位）
- **<60分**：过滤掉

## 系统使用提醒

### 运行前检查
1. ✅ 填写 `config/user_profile.json` 中的联系方式
2. ✅ 在 `config/.env` 中设置 OpenAI API Key
3. ✅ 配置飞书/钉钉 Webhook（可选）
4. ✅ 简历PDF放在指定路径

### 首次运行
```bash
# 显示浏览器，方便扫码登录
python main.py --no-headless --platforms boss zhaopin
```

### 日常使用
```bash
# 后台运行
python main.py

# 定时任务（每天早上9点）
python scheduler.py
```

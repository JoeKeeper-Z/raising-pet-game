# 星尘物语 开发日志

## 2025-05-26 重大更新：外出探索系统重构

### 新增功能

#### 1. 美少女梦工厂风格外出探索系统
- **全屏沉浸式场景设计**：每个地点有独特的渐变背景和氛围描述
- **NPC立绘卡片**：精美的人物展示，带NEW标识和好感度爱心
- **场景装饰**：大型半透明emoji营造氛围

#### 2. 好感度系统
- **7个关系等级**：陌生人 → 点头之交 → 普通朋友 → 好朋友 → 亲密挚友 → 至交 → 知己
- **互动方式**：
  - 💬 闲聊：基础对话+5好感
  - 📖 深入交流：话题讨论
  - 🎁 赠送礼物：鲜花(+10)、书籍(+15)、茶叶(+20)、饰品(+30)
  - 🌸 共度时光：好感度≥100解锁，+20好感

#### 3. 防刷机制
- **探索次数限制**：每个地点每天最多探索2次
- **NPC对话限制**：
  - 每次外出最多与3个不同NPC对话获得属性
  - 每个NPC每次外出只能获得1次属性
- **次数显示**：按钮实时显示剩余次数

#### 4. 属性数值平衡
- 大幅降低NPC对话的属性收益（原来2-3点 → 现在1点）
- 探索奖励调整为小数值（+1感知/+3心情等）
- 送礼成本高，但收益合理

#### 5. 视觉优化
- 外出面板深色背景+金色字体
- 地点卡片悬浮动画效果
- NPC卡片滑入动画
- 属性变化即时浮窗提示

## 2025-05-25 代码模块化重构

### 文件拆分
原 `星尘物语.html` (5842行) 拆分为：
- `css/style.css` - 游戏样式
- `js/modules/data.js` - 游戏数据
- `js/modules/events.js` - 事件系统
- `js/modules/dialog.js` - 对话系统
- `js/modules/core.js` - 核心逻辑
- `js/modules/shop.js` - 店铺系统
- `js/modules/ui.js` - UI交互
- `index.html` - 主入口

### 架构改进
- 模块化JavaScript架构
- window全局变量管理游戏状态
- 各模块职责分离，便于维护

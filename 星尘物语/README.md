# 星尘物语 (Stardust Story)

一款类似《美少女梦工厂》的养成类游戏，玩家扮演父亲/养父，培养女儿星儿成长，经历各种事件，最终达成不同结局。

## 游戏特色

### 🎮 核心玩法
- **三年养成周期**：36个月的培养时间，见证女儿从幼年到成年的成长
- **8种基础属性**：体力、智力、魅力、社交、气质、道德、力量、感知，影响结局走向
- **多结局系统**：S/A/B/C级结局，根据属性解锁不同人生道路
- **事件链系统**：选择影响后续事件发展，每个选择都有意义

### 🗺️ 外出探索系统（新增）
- **8个可探索地点**：学校、图书馆、集市、武道馆、贵族区、马厩、森林、广场
- **NPC好感度系统**：
  - 7个关系等级（陌生人→点头之交→普通朋友→好朋友→亲密挚友→至交→知己）
  - 4种互动方式：闲聊、深入交流、赠送礼物、共度时光
  - 多位可结识NPC，每位都有独特剧情
- **防刷机制**：
  - 每个地点每天最多探索2次
  - 每次外出最多与3位NPC对话获得属性
  - 平衡的属性收益，避免无限刷属性

### 💬 对话系统
- **父女对话**：每月可与女儿进行多轮对话
- **话题多样**：根据游戏月份变化，包含节日、成长烦恼等
- **选择影响**：不同回应影响属性和好感度

### 🏪 店铺经营
- **多种店铺类型**：花店、书店、道具店等
- **经营管理**：进货、定价、促销策略
- **收益系统**：经营获得金币支持培养

### 📅 日程安排
- **每月计划**：自主安排女儿的学习和活动
- **活动效果**：不同活动影响不同属性
- **日程演出**：生动的活动场景展示

### 🎭 事件系统
- **随机事件**：探索中触发各种意外事件
- **主线剧情**：完整的成长故事线
- **事件记忆**：记录玩家的选择历史

## 运行方式

直接在浏览器中打开 `index.html` 即可运行游戏。

**注意**：由于浏览器CORS限制，建议通过本地服务器运行：
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`
